## Manager access (Nur's question answered)

| Platform | How to add a manager |
|---|---|
| **Facebook + Instagram** | Meta **Business Suite** → Settings → People → add `info@sinan.agency` → assign **Admin** (or Editor) on each Page/IG. (Business invites already accepted ✅.) |
| **LinkedIn (Page)** | Page → Admin tools → Manage admins → add by name/email. |
| **TikTok** | TikTok **Business Center** → add member → assign asset roles. |
| **Pinterest (Business)** | Settings → Account access → add via email. |
| **YouTube** | Channel via a **Brand Account** → add manager in Google account settings. |
| **X (if used)** | Delegate via X Pro / share via password manager. |

> Best practice: brands owned by a **Business/Brand account**, not a personal profile, so access survives staff changes (mirror of the Drive principle).
